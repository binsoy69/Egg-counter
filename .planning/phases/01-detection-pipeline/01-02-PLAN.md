         - Calls classify_by_ratio(egg_bbox, self.zone_config, self.nest_box_width_mm)

  <action>
    Create src/egg_counter/tracker.py with class EggTracker:

    1. __init__(self, stability_seconds: float = 3.0):
       - self.stability_seconds = stability_seconds
       - self.counted_ids: set[int] = set()  # Track IDs already counted
       - self.pending_tracks: dict[int, float] = {}  # track_id -> first_seen_timestamp
       - self.active_tracks: set[int] = set()  # Currently visible tracked eggs in zone
       - self.had_eggs = False  # Whether eggs were present (for collection detection)

    2. process_detections(self, track_ids: list[int], boxes: list, in_zone_flags: list[bool], timestamp: float) -> list[dict]:
       - Takes parallel lists: track_ids, bounding boxes (xyxy), boolean flags for zone containment, and current timestamp
       - Returns list of event dicts (empty list if no new events)
       - For each (track_id, box, in_zone) tuple:
         a. If in_zone and track_id not in self.counted_ids:
            - If track_id not in self.pending_tracks: add to pending_tracks with current timestamp
            - Elif timestamp - self.pending_tracks[track_id] >= self.stability_seconds:
              - Add track_id to self.counted_ids
              - Remove from self.pending_tracks
              - Append {"action": "count", "track_id": track_id, "bbox": box} to results
         b. If not in_zone and track_id in self.pending_tracks:
            - Remove from pending_tracks (timer reset per D-04)
       - Update self.active_tracks = set of track_ids that are in_zone OR in self.counted_ids (for occlusion: keep counted eggs as "active" even if not currently detected per D-06)
       - Check collection: if self.had_eggs and len(current_in_zone_ids) == 0 and len(track_ids) == 0 and len(self.counted_ids) > 0:
         - Emit {"action": "collected", "count": len(self.counted_ids)}
         - Clear self.counted_ids, self.pending_tracks, self.active_tracks
         - self.had_eggs = False
       - If any eggs are in zone: self.had_eggs = True
       - Return results list

    3. initialize_from_existing(self, track_ids: list[int]) -> None:
       - Per D-08: On restart, mark visible eggs as already counted
       - Adds all provided track_ids to self.counted_ids
       - Sets self.had_eggs = True if track_ids is not empty
       - Does NOT emit any events

    4. Property egg_count -> int: returns len(self.counted_ids)
