    Create src/egg_counter/preview.py with two public functions:

    1. `draw_detections(frame, detector_result, zone_config, classifier, egg_count)` -> annotated frame:
       - Draw the zone rectangle in green (cv2.rectangle with zone_config x1,y1,x2,y2, color=(0,255,0), thickness=2)
       - For each detection in detector_result["boxes"]:
         - Draw bounding box rectangle in blue (0,165,255) if is_in_zone, gray (128,128,128) if outside
         - Classify size using classifier.classify(box) -> (size, height_mm)
         - Draw label text above box: f"#{track_id} {size} {conf:.0%}" using cv2.putText, font FONT_HERSHEY_SIMPLEX, scale 0.6, white text with black outline for readability
         - Draw height_mm below box: f"{height_mm:.1f}mm"
       - Draw egg count overlay in top-left corner: f"Eggs: {egg_count}" in large font (scale 1.2), white with black outline
       - Return the annotated frame (modify in-place is fine, numpy array)

    2. `run_preview(model_path, camera_index=0, video_path=None, config_path="config/settings.yaml", zone_path="config/zone.json")`:
       - Load settings via load_settings(config_path), zone via load_zone_config(zone_path)
       - Create EggDetector(model_path, tracker_config=settings.get("bytetrack_config", "config/bytetrack_eggs.yaml"), confidence=settings.get("confidence_threshold", 0.5))
       - Create SizeClassifier(zone_config)
       - Open camera or video using same platform-aware pattern as pipeline.py run() method
       - Track egg_count as integer, incremented when a new track_id is seen (use a set of seen_track_ids)
       - Main loop:
         - Read frame from capture
         - Run detector.detect_and_track(frame)
         - For each track_id in result that is_in_zone and not in seen_track_ids: add to seen_track_ids, increment egg_count
         - Call draw_detections(frame, result, zone_config, classifier, egg_count)
         - cv2.imshow("Egg Counter Preview", annotated_frame)
         - key = cv2.waitKey(delay) where delay = int(1000/frame_rate) for video, 1 for camera
         - Break on 'q' key or ESC (key == ord('q') or key == 27)
       - Cleanup: cap.release(), cv2.destroyAllWindows()
       - Print summary: f"Preview ended. Total eggs seen: {egg_count}"
