    1. Create src/egg_counter/logger.py with class EggEventLogger:
       - __init__(self, log_dir: str = "logs"): stores log_dir, creates directory if not exists, initializes egg_count = 0
       - log_egg_detected(self, track_id: int, size: str, confidence: float, bbox: list, size_method: str, raw_measurement_mm: float, frame_number: int) -> dict:
         - Increments self.egg_count
         - Builds event dict per D-15: {"type": "egg_detected", "timestamp": ISO 8601 UTC string (datetime.now(timezone.utc).isoformat()), "track_id": track_id, "size": size, "confidence": round(confidence, 3), "bbox": bbox (list of 4 floats), "size_method": size_method, "raw_measurement_mm": round(raw_measurement_mm, 1), "frame_number": frame_number}
         - Calls _write_event(event)
         - Prints to stdout per D-16: f"New egg #{self.egg_count} -- {size}"
         - Returns the event dict
       - log_eggs_collected(self, count: int) -> dict:
         - Builds event: {"type": "eggs_collected", "timestamp": ISO 8601 UTC, "count": count}
         - Calls _write_event(event)
         - Resets self.egg_count = 0
         - Prints: f"Eggs collected: {count}"
         - Returns event dict
       - _write_event(self, event: dict): Opens today's log file in append mode. Filename: f"eggs-{datetime.now().strftime('%Y-%m-%d')}.jsonl". Writes json.dumps(event) + "\n". Per D-17 (daily rotation) and D-18 (egg events only).
       - _get_log_path(self) -> Path: Returns Path(self.log_dir) / f"eggs-{datetime.now().strftime('%Y-%m-%d')}.jsonl"
